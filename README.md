<h1>📁 KeenKeeper — Personal Interaction & Activity Tracker</h1>

<img width="1032" height="804" alt="Screenshot 2026-04-16 233829" src="https://github.com/user-attachments/assets/791d8896-7206-4085-b6a0-ca1a6f29039c" />

<p>
KeenKeeper is a modern web application that helps you track and manage your interactions with friends and family. Whether it's a phone call, text message, or video chat, it keeps your social connections organized and easy to monitor.
</p>

<img width="1292" height="714" alt="Screenshot 2026-04-16 233846" src="https://github.com/user-attachments/assets/42308f5e-a0ae-4eb9-b526-1318f476041f" />


<h2>🚀 Key Features</h2>

<ul>
  <li>Dynamic dashboard rendering contact cards from structured JSON data</li>
  <li>Dedicated dynamic routes for each contact with personalized insights</li>
  <li>Real-time activity logging (Audio Call, Text, Video) using Context API</li>
  <li>Smart filtering of interaction history by activity type</li>
  <li>Data visualization with a donut chart using Recharts</li>
  <li>Fully responsive design across mobile, tablet, and desktop</li>
  <li>Real-time user feedback using React Toastify notifications</li>
</ul>

<img width="1860" height="668" alt="Screenshot 2026-04-16 233907" src="https://github.com/user-attachments/assets/524f8028-164c-4e5f-83ae-b9c1c15930c7" />


<h2>🛠️ Tech Stack</h2>

<ul>
  <li>Framework: Next.js (App Router)</li>
  <li>State Management: Context API</li>
  <li>Styling: Tailwind CSS & DaisyUI</li>
  <li>Charts: Recharts</li>
  <li>Notifications: React Toastify</li>
  <li>Icons: React Icons</li>
</ul>

<hr/>

<h2>💡 Overview</h2>

<p>
KeenKeeper is built with a focus on simplicity, scalability, and user experience. It leverages modern React patterns like the Context API and dynamic routing to deliver a smooth and interactive experience. The application makes it easy to maintain, track, and visualize personal communication habits in one place.
</p>
